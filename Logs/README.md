# Experimental Logs

Lab notebook for our glue project. Each numbered entry (`NNN_*.md`) is a dated,
self-contained record of one experiment: objective, method, key scripts,
results, and conclusions.

Where the rest lives (kept in one place each, to avoid drift):

- **Project context, goals, terminology — and the experiment index** (what each
  entry found, with links): [`docs/RESEARCH_CONTEXT.md`](../docs/RESEARCH_CONTEXT.md).
- **Repo layout + where datasets and result files live**:
  [`docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md).
- **Paper shelf** (method + biology, cite-by-key): [`references/`](references/).

Read `docs/RESEARCH_CONTEXT.md` before writing a new entry, and use the
`experiment-log` skill to scaffold it.
